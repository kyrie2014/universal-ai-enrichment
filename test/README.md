## 🚀 快速测试

### 1. 企业信息补充（推荐）

使用 `test_data/测试数据_企业信息.xlsx` 测试企业信息查询功能：

1. 启动程序：`python agent_main.py`
2. 选择方案：`企业信息补充`
3. 上传文件：`examples/test_data/测试数据_企业信息.xlsx`
4. 指定输入列：`公司`
5. 点击"开始处理"

### 2. 产品信息补充

使用 `test_data/测试数据_产品信息.xlsx` 测试产品信息查询功能。

### 3. 人物信息补充

使用 `test_data/测试数据_人物信息.xlsx` 测试人物信息查询功能。

## 🛠️ 测试数据生成

如果需要生成新的测试数据：

```bash
cd examples
python create_test_data.py
```

这将在 `test_data/` 目录生成新的测试文件。

## 🎯 自定义测试

### 创建自己的测试数据

1. 复制现有的测试Excel文件
2. 修改输入列数据
3. 在程序中创建或编辑方案
4. 定义需要补充的字段
5. 运行测试

### 示例：创建"书籍信息补充"方案

1. 创建Excel文件，列名为"书名"
2. 在程序中点击"新增方案"
3. 设置：
   - 方案名称：书籍信息补充
   - 输入列：书名
   - 输出列：作者、出版社、ISBN、出版日期、简介
4. 编辑提示词模板
5. 保存并测试
